🚀 快速上手

完整流程，按顺序排列。一次性设置请在家用 WiFi 完成。

① 立即可用（无需设置）：卫星照片

卫星照片 — 地面的真实影像，无需账户。用于查看裸露／多岩区域、河流、道路、旧矿坑与进入路线。偏红地面提示会粗略突出锈色斑块（可能含铁——但须实地核查）。

② 可选升级：真正的分析图层 — 完整设置指引

这将解锁真正的铁／黏土／雷达波段运算。它是免费的，但 Copernicus 网站较为混乱，请严格按以下步骤操作。请在家用 WiFi 完成，约需 20 分钟。只需做一次。

第一部分 — 创建账户
大多数人卡在这一步。Copernicus 经常改版网站，且没有明显的"注册"按钮。不要直接访问仪表板、账户或登录网址——在还没有账户时，那只会把你弹到登录页面，且无处注册。

1. 前往 dataspace.copernicus.eu
2. 先试首页上的蓝色 Get started 按钮——在当前版面下这最可能通向注册。
3. 若无效，点击右上角 LOGIN → Your profile。这会打开登录页面，表单上通常有 Register ／ Create account 链接。
  注意：LOGIN 下拉菜单本身没有注册选项，它只列出可登录的去处（Your profile、openEO、Sentinel Hub）。不要在那里找 Register。
4. 填写必填项，勾选接受条款与条件及隐私政策，然后提交。
5. 打开验证邮件并点击 "Verify email address"。在此之前账户不会生效——这是最常被遗漏的一步。

遇到问题？Copernicus 支持邮箱：help-login@dataspace.copernicus.eu

捷径 — 可跳过整个第一部分：如果已有人设置好实例 ID，直接用他的即可。一个 ID 可用于多部手机，能完全避开这整个流程。只有当你想要自己的配额时才需要自行注册。

有了账户之后，回到仪表板最快的方式是从首页点击 LOGIN → Sentinel Hub。

第二部分 — 获取实例 ID
1. 前往 shapps.dataspace.copernicus.eu/dashboard/#/configurations
2. 用第一部分创建的账户登录
3. 你会看到配置列表。可使用现成的 "Simple WMS Instance"，或点击 New configuration，命名（例如 ProspectField），并基于 Simple WMS template 创建
4. 进入该配置。实例 ID 就是显示的那串长代码（形如 c32c21b1-84ac-…）。它可能出现在类似 .../wms/那串代码 的网址中——你只需要 /wms/ 之后的代码，而不是整个网址
5. 复制它 → 在本应用中进入地图页 → 展开 🔬 高级分析图层 → 粘贴到输入框
6. 点击 🔎 测试我的 ID。若显示出影像，即表示连接成功。

第三部分 — 添加图层
在你的配置中，为下列每一项点击 ＋ New layer。名称须完全一致（大小写与连字符都重要——应用按这些名称查找），选好数据源后保存。

• SENTINEL-1 — 数据源：Sentinel-1 GRD。若提示"请选择预定义产品或输入处理脚本"，随便选一个预定义产品即可（例如 VV 或 VV+VH decibel gamma0）。这就是你的雷达图层。
• IRON-RATIO — 数据源：Sentinel-2 L2A，Data Processing → Custom script → 粘贴铁比值脚本
• FERROUS-IRON — 数据源：Sentinel-2 L2A → Custom script → 粘贴亚铁脚本
• ALTERATION — 数据源：Sentinel-2 L2A → Custom script → 粘贴蚀变脚本

三个自定义脚本在下一节，请分别复制到对应图层。

如果无法正常工作
• 点击图层没有任何反应 — 你缩放得太远了。卫星数据不会在全国尺度上提供；请放大到城镇大小的范围。
• 光学图层空白但雷达正常 — 那是云层，在马来西亚很正常。雷达可以穿透云层。
• 错误 400 — 通常是图层名称不完全一致。请对照上面的列表检查大小写和连字符。
• 测试我的 ID 失败 — 检查 ID 是否只有代码本身，没有空格或多余的网址文本。

②b 三个自定义脚本（复制到 Copernicus）

请把每个脚本粘贴到 Copernicus 配置工具中对应图层的 Custom script 框内。

IRON-RATIO — 亮红色＝锈色、铁染地面与铁帽

FERROUS-IRON — 第二种、不同的铁矿物信号

ALTERATION — 红＝铁，绿＝黏土，蓝＝亚铁。偏红黄＝铁＋黏土同时出现，即典型的蚀变标志

点击复制按钮，然后粘贴到 Copernicus 中对应图层的自定义脚本框。（也可以点击脚本框手动选取。）

为什么这些比普通波段比值更清晰。在马来西亚，丛林覆盖了大部分地面，而植被的反射特性会让普通的铁比值到处发红——整幅影像变成浑浊的中红色雾霾，真正的目标淹没其中。这些脚本会先屏蔽植被和水体（丛林渲染为平淡的深灰绿，水体为深蓝），只对裸露地面着色，并采用更紧凑的对比拉伸。

解读方法：深灰绿＝丛林，忽略。深蓝＝水体。鲜明色彩＝有真实信号的裸露地面。若整屏几乎全暗，说明该区域没有岩石出露——可改用「植被空隙」图层找出哪里有。
☁️ 关于云层——以及为什么旧影像也没问题。Sentinel-2 大约每 5 天过境一次，本应用会回溯搜索 2 年内的影像。因此关键不是今天的天气，而是过去是否有任何一次过境拍到晴空。你可能头顶蓝天却拿到多云影像，也可能在雷雨中扫描却看到几个月前的晴空影像。

旧影像对勘探不是问题。数百万年前形成的矿床不会移动——较旧的无云影像与最新影像显示同一片岩石，而且更清晰。这正是本应用默认仅晴空影像（≤10% 云量）的原因。若某处始终加载不出，可切换到允许多云，或改用完全不受云层影响的雷达。

马来西亚较干燥的月份晴空过境更多，那些时段的影像库更丰富。
③ 在家用 WiFi：从太空侦察

在地图页平移到你伙伴的执照区块。开启高级图层后：铁与黏土显示可能的目标，雷达穿透云层显示断层，植被空隙找出可实地查看的裸露地面。记下有希望的位置。（还没有 Copernicus ID？普通卫星照片同样有助于规划。）

④ 仍在 WiFi 时：下载地图

同样在地图页 → 平移／缩放到目标区域 → 下载当前视图供离线使用。这样地图在野外没有信号时也能用。每个你打算徒步前往的区域都要下载一次。

⑤ 出发前：告诉一个人

轨迹页 → 设定联系人的 WhatsApp 号码、姓名和返回时间 → 在仍有信号时（通常在车辆旁）点击「我出发了」。这是最重要的一步。然后点击开始记录轨迹，并选择均衡省电模式。

⑥ 现场：评估、取样、拍照

在有希望的位置，用评估页打分。值得取样？前往取样页 → 记录 GPS、添加照片、勾选你看到的标签。高分位置就是之后要送实验室化验的对象。

首次出野外前请先阅读下方「🔨 如何正确取样」。沿露头刻槽取样而不是随手捡一块漂亮石头，正是有意义的化验与误导性化验之间的差别。

⑦ 撤离：原路返回并报平安

轨迹页 → 原路返回起点会沿着你的面包屑路径准确带你回去。回到车辆并恢复信号后，点击「我已安全出来」让联系人解除警戒。

⑧ 之后：送实验室确认

卫星线索和野外评分都不是证据。把最好的样品送到有资质的实验室（JMG 或商业化验所）。结果为正 → 再聘请持证勘探地质师，并与你的伙伴一同遵循官方程序。

⚠ 云层提醒。马来西亚常年的云层经常遮挡光学图层（铁、黏土、植被）。雷达始终可用。若某个光学图层看起来空白，那是云层——请改用雷达或改天再查。

🎒 基本装备清单

想跟着做的人需要什么。这份清单按优先级排列——先买上面的。多数东西在五金店或网上就能买到，不必买昂贵的专业货。

① 必备 — 没有这些别出门

地质锤（约 RM80–200）——尖头或凿头款。这是唯一真正专业的工具，值得买好的；普通羊角锤会碎裂伤人。

手持放大镜 10 倍（约 RM20–60）——辨认矿物的关键。10 倍是标准；不要买倍率更高的，视野太窄不好用。

护目镜（约 RM15）——敲石头会崩碎片。眼睛只有一双。

结实手套（约 RM15）——防割伤，也防某些矿物。

样品袋——厚塑料袋或布袋，一样一袋，绝不重复使用。

永久记号笔——在袋上写编号。

充电宝 10000mAh（约 RM40–80）——手机是你的 GPS、地图和求救工具。

实体指南针（约 RM30–60）——手机会没电，这是保命的备份。请买带度数刻度、可旋转表圈的底板式指南针（Silva 或 Suunto 那类），不要钥匙扣式小指南针——你需要能设定一个方位并沿着它走。导航页会直接给出要设定的度数。

② 很有用 — 尽快添置

淘金盘（约 RM30–60）——塑料的比金属好用（较轻、有沟槽）。检验河床沉积物的基本工具。

小铲或手镐——采集土壤和沉积物样品。

筛子（约 5mm 和 2mm）——淘洗前筛除粗砾。

强力磁铁（约 RM10）——检验磁铁矿；黑砂中吸出磁性组分。

稀盐酸（10%，小滴瓶）——滴在石灰岩上会起泡，用于识别碳酸盐岩。请小心携带并标明。

硬度测试组——一枚铜币（硬度约 3.5）、一把小刀（约 5.5）、一小块石英（7）。用来快速判断矿物硬度。

卷尺或折叠尺——量脉宽，也可作照片比例参照。

野外记录本（防水纸更好）——手机会没电，纸不会。

③ 安全装备 — 独自入林必备

独自在丛林中，这些和地质工具同样重要：

头灯加备用电池——赤道地区天黑得又快又早（约 19:00）。

急救包——含绷带、消毒剂、止血敷料。

哨子——比喊叫传得远，也更省力。

足量饮水加净水片或滤水器。

驱蚊液（含 DEET）——登革热和疟疾是真实风险。

雨衣和防水袋——保护手机与样品。

结实靴子加长裤——防蛇、防蚂蟥、防割伤。

蚂蟥袜——马来西亚雨林中非常值得。

如果经常独自深入无信号区，可考虑卫星求救器（如 Garmin inReach）。价格不菲，但那是唯一在没有手机信号时仍能呼救的东西。

④ 之后再考虑（别先买）

盖革计数器／闪烁仪（约 RM400–1500）——只有认真找稀土或钍时才需要。

紫外灯——少数矿物有荧光（白钨矿等），用途有限。

手持 XRF 分析仪（数万令吉）——专业设备。不要买；需要时可租，或直接送实验室。

比重瓶——测密度，实验室做更准。

坦白说：大多数新手在装备上花太多、在化验上花太少。一把好锤子、一个放大镜和三次实验室化验，胜过一柜子从没用过的仪器。

先从必备清单开始。全部加起来大约 RM250–450，就能让你正确取样并安全归来。真正的花费在化验上（每个样品约 RM80–250），而那才是唯一能告诉你岩石里有什么的东西。装备帮你取到好样品；化验才给出答案。

🎓 从这里开始 — 勘探是怎么回事

如果你是新手，请先读一遍。它解释了我们为什么要找这些东西。

一个核心概念，解释一切

有价值的金属——金、银、铜、铅、锌——通常在地下深处以硫化物矿物形式形成（金属与硫结合）。热液沿岩石裂隙将它们带上来并沉淀于此。

你几乎从不会直接看到金属本身。你要找的是这些热液留下的副产物：石英脉、蚀变岩石和锈色染色。勘探是侦探工作，不是寻宝。

为什么锈色地面（氧化铁）如此重要

硫化物在地表不稳定。历经数千年，雨水与氧气将其分解，其中的铁转变为锈色氧化铁，作为风化壳残留下来。这层壳称为铁帽。

因此推理链是：锈色染色 → 此处曾有硫化物 → 硫化物常携带黄金或其他金属 → 值得调查。

这是最古老的找矿线索之一，在任何技术出现之前已沿用数百年。它也能从太空看到，这正是本应用设有氧化铁卫星图层的原因。

⚠ 但大多数锈色岩石只是锈色岩石

这是几乎每个新手都会犯的错误。铁是地球上最常见的元素之一，因此红棕色染色随处可见，原因却完全平常——普通含铁矿物风化、热带红土（马来西亚极为常见）或单纯的铁锈。

真正的铁帽具有普通染色所没有的特征：
• 网格状／海绵状构造——硫化物晶体溶解后留下的蜂窝状小孔洞。这是关键判据。普通锈色土壤是致密的；铁帽看起来呈细胞状，近似海绵或蜂巢。
• 伴生石英脉——位于脉体之上或旁侧，而非均匀铺满整个山坡。
• 混有异常颜色——绿或蓝（铜）、煤烟黑（锰）、黄色。多色比单一红色更有意义。
• 手感偏重——常比外观显得更致密。

仅凭鲜艳颜色毫无意义。覆盖整个坡面的均匀红棕色几乎总是红土。

真正的准则：看多项证据是否一致，而非单一线索

单一指标本身意义不大。真正重要的是多项指标在同一地点指向同一结论：

铁染 + 石英脉 + 断层或岩性接触带 + 硫化物痕迹

这样的组合才值得正式取样并送化验。一块锈色岩石则不值得。

这正是"评估"页需要勾选多项指标才会给出高分的原因——它的设计初衷就是阻止你追逐单一线索。

什么才能"确认"发现（什么不能）

你能看到的一切都无法确认任何事。卫星图层不行，野外指标不行，本应用的高分也不行。它们只做一件事：把数千公顷缩小到几个值得取样的地点。

只有实验室化验（ICP 或火试金）才能告诉你金属是否真实存在、含量多少。而且即使化验结果良好，那也只是一个点——证实一个矿床需要系统取样，最终还需钻探，并有持证地质师参与。

这样理解：本应用帮你决定把取样经费花在哪里，它无法告诉你岩石里有什么。

你会反复见到的术语

矿石——含有足够可采金属、值得开采的岩石。
硫化物——金属与硫的化合物；地下大多数金属以此形式存在。
铁帽——硫化物氧化后残留的锈色风化壳。
蚀变——岩石被热液化学改造，常富含黏土且颜色较浅。
脉——被矿物充填的裂隙，常见为石英。
露头——出露地表且仍在原位的基岩。
转石——已从原生位置移动的松散岩块；是线索，但不能证明其下方有什么。
接触带——两种岩性相接之处；成矿的有利部位。
化验——测定样品中金属含量的实验室检测。
砂矿——被水流富集于河床砾石中的金属。
找矿标志——容易发现、指示更难察觉目标的线索。
斑岩——以侵入体为中心的大型低品位铜／金系统，周围有大范围蚀变。
矽卡岩——侵入体与石灰岩反应形成的矿床。
网脉——密集的细脉网络，而非单一大脉。

所有功能在世界任何地方都可使用：GPS、取样、照片、离线地图、面包屑轨迹、指南针、安全报平安，以及全部卫星图层（Sentinel-1 与 Sentinel-2 覆盖全球）。

属于马来西亚特有、在别处需要调整的是：区域实例（中央金带、近打谷、石隆门、马穆特）、对锡与热带红土的侧重，以及最重要的执照法规——各国矿业法完全不同。标记为「区域特定」的部分在国外使用时须格外留意。

📖 马来西亚野外指标 — region-specific

地表线索及其在马来西亚地质中所指示的矿种。

铁帽 / 铁质盖层

→ gold, base metals

锈红棕色风化岩石（氧化的硫化物）。覆盖埋藏矿化之上的经典找矿指示。应对其本身及附近石英脉取样。
与普通铁锈的区别：真正的铁帽具有海绵状网格构造——硫化物晶体溶解后留下的小孔洞——且通常伴生石英脉。整片坡面均匀的红棕色通常只是红土。

花岗岩／片岩接触带的石英脉

→ lode gold, tungsten

中央金带（彭亨–吉兰丹–登嘉楼）的金赋存于靠近花岗岩边缘的石英脉中。寻找乳灰色、多孔的石英。

河床捕获处的黑色重砂

→ tin (cassiterite), gold, monazite (REE)

淘洗其精矿。冲积锡石分布广泛；黑砂组分中的独居石则含稀土。

风化花岗岩 / 红土

→ ion-adsorption REE

深度风化的花岗岩可将稀土富集于黏土中。肉眼无可靠特征——应采集富黏土腐岩并送化验。

石灰岩／大理岩接触带（矽卡岩）

→ iron, copper, gems

侵入体与石灰岩的接触带（近打谷、浮罗交怡）可赋存矽卡岩矿物和刚玉。

方铅矿与硫化物脉

→ silver (with lead & zinc)

破裂成立方体的银灰色重方铅矿是主要的白银载体——白银很少以可见金属形式出现。寻找布满金属硫化物的石英脉、条带／壳状构造，以及黑色锰染。在马来西亚，白银通常是多金属系统的副产品，而非独立目标。

绿色与蓝色染色

→ copper (porphyry or skarn)

鲜绿的孔雀石与深蓝的蓝铜矿是风化的铜——所有金属指示中最醒目的，常在数米外即可发现。另留意黄铜色的黄铜矿、虹彩的"孔雀铜"斑铜矿，以及网脉（密集的细石英脉网）。
规模很重要：大部分铜产于大型低品位斑岩系统，因此大面积蚀变带加浸染状硫化物，胜过一块看起来很富的岩石。沙巴的马穆特即属此类。

⚠ 现实提醒。以上都不能确认经济矿床。它们只告诉你该去哪里取样送化验（ICP／火试金）。野外工作缩小目标——化验和钻探才能证实。

🌍 卫星照片／真彩色

→ the lie of the land

真实影像。用于识别裸岩、河流、道路、旧矿坑及进入路线。你的规划底图。

🟠 铁比值

→ gossans, iron staining (gold pathfinder)

亮红橙色＝锈色铁染地面。铁帽覆盖许多金与贱金属矿床，因此明亮斑块是优先取样点。

🔶 亚铁

→ a second iron signature

捕捉与铁比值不同的铁矿物。当两者在同一位置同时发亮时，信号更强。

🟫 黏土／短波红外 与 🌈 蚀变合成

→ hydrothermal alteration

黏土与蚀变岩石环绕许多矿体。在合成图上，偏红黄＝铁与黏土同时存在，即经典蚀变标志。应查看铁与黏土重叠之处。

📡 雷达

→ faults & structure (vein control)

可穿透云层。笔直的明暗线条＝山脊与断层。金脉常沿断层分布，因此应追索这些线性构造——尤其在它们穿过铁／蚀变斑块之处。

🌿 植被空隙

→ where rock is exposed

橙／棕褐色＝你实际可以查看的裸露或多岩地面。绿色＝丛林遮蔽了地质。

最强的目标是多个图层一致指向的位置：铁＋黏土斑块，位于或靠近雷达断层线，且地表裸露。那就是应优先取样的地点。在 20 米分辨率下，这些只能找到矿区而非单条脉；只有实验室化验才能确认。

🔨 如何正确取样

取样不当会产生毫无意义的化验结果。你可能其他都做对了，却在这里白花钱——首次出野外前值得一读。

最大的一个错误

捡起一块漂亮的石头。

单块看起来富矿的标本几乎说明不了任何事。矿化是不均匀的——你挑最亮的一块，化验结果反映的就是那块幸运石头，而不是这片地。很多人因此被严重误导，有些还为此花了真金白银。

你真正需要的是代表性样品：能公平反映整个露头的物料，好的和普通的一起取。

岩石／露头 — 刻槽取样

不要只取一块。正确做法：

• 沿露头按固定间隔敲取小碎块——例如横跨脉体或矿化带每 20–30 厘米取一处。
• 总量约 1–2 公斤。少于此实验室可能不够用；多得多则只是白白负重。
• 把普通物料与好看的部分一起取。刻意避开无趣的岩石，就是在骗自己。
• 横跨脉体取样（垂直方向），而非沿脉走向——你要知道的是整个脉宽的品位。
• 尽量敲取新鲜岩石。风化面上的金属可能已被淋失或次生富集，两个方向都会误导你。

河流沉积物 — 普查主力

一个河流样品代表上游侵蚀下来的全部物质，因此是高效试探大面积的手段。

• 取活动沉积物，即有水流处，而非静止的岸边。
• 优先选择河湾内侧、巨石后方和壶穴——重矿物在此沉降。
• 在数米范围内取多处合并成一个样。
• 筛除粗砾，你要的是较细的组分。
• 记录它来自哪条支流。若某个样品异常，就从该汇合口向上游追索缩小源区。

土壤取样

• 挖穿枯枝落叶层与表土——取下方的 B 层（通常 20–50 厘米），金属在此聚集。
• 各样品之间深度保持一致，否则结果无法比较。
• 约 500 克足够。
• 在坡地上要记得物质会向下蠕移——异常的源头通常在上坡方向。

淘洗

• 从重矿物富集处装满淘金盘，浸入水中，以画圈动作并前倾把轻组分淘出。
• 保留黑砂——这一重组分含金、锡石和独居石。
• 淘洗是野外指标，不是化验。它说明有东西存在、值得追索，但不能测定品位。
• 记录你看到的"金点"数量——这是不同地点之间的粗略比较。

避免污染

• 使用干净样品袋，一样一袋，绝不重复使用。
• 各点之间清洗淘金盘和工具，尤其在取过富矿样品之后。
• 保持样品袋封闭——来自其他地点或车辆的粉尘会扭曲低品位结果。
• 除非那正是你的目标，否则避免紧贴路堑、旧矿废石堆或围栏线取样；这些位置常有外来物料混入。

📷 正确拍照记录

没有比例参照的岩石照片，对日后查看的人几乎毫无用处——包括三个月后的你自己。

• 务必在画面中放入比例参照物：硬币、地质锤、笔或镜头盖。硬币最便于携带，而且人人都认得其大小。
• 拍三张：较广的环境照（能看到露头及其背景）、目标特征的特写、以及贴有编号标签的样品本身。
• 同时拍摄新鲜断面和风化面——两者外观完全不同，新鲜面才显示真实矿物组成。
• 尽量在柔和光线下拍摄。正午强光的阴影会掩盖结构；稍微把岩石淋湿可突出颜色与构造。
• 在笔记中记下你面朝的方向，以便日后把照片与实地对应起来。

标记与保密

使用本应用生成的样品编号，用记号笔把同一编号写在样品袋上，并拍下贴好标签的袋子。

不要把位置写在袋子上。仅凭编号实验室就能返回结果——坐标留在你手机里的本应用中。这样下游经手样品的任何人都无法得知它来自何处。

把编号与位置的私人对应关系保存在应用内，并做好备份，以免丢失这一联系。

为什么这很重要。化验结果的好坏取决于样品。随手抓取的样品可能读数偏高、让你追逐虚幻，也可能读数偏低、让你放弃真正有价值的东西。规范取样是避免这两种错误最省钱的办法。

🗺️ 马来西亚矿产区 — 区域特定

以往在哪里有过发现，以及应预期何种地质。马来西亚半岛与东马来西亚确有不同——找什么取决于你在哪里。

中央金带 — 吉兰丹、登嘉楼、彭亨

沿半岛东侧南北延伸的地带，是马来西亚主要产金区。金赋存于石英脉中，由沿断层运移、靠近花岗岩边缘的热液形成。

已知矿山包括彭亨的 Penjom、Selinsing 和自殖民时期即开采的 劳勿（Raub）。

寻找：乳灰色石英脉（尤其多孔或带锈者）、铁帽、硫化物矿物，以及花岗岩–片岩接触带。

近打谷 — 霹雳

怡保一带的历史锡矿腹地，曾是全球最富的锡矿区之一。锡来自冲积砾石中的锡石以及花岗岩–石灰岩接触带。怡保著名的石灰岩山丘属于同一套地质。

至今仍有意义：旧锡矿尾砂可能含独居石，这种稀土矿物被早期矿工当作废料丢弃。

寻找：黑色重砂、锡石颗粒、冲积砾石、旧矿坑与尾砂。

浮罗交怡与石灰岩区 — 吉打、玻璃市

石灰岩与大理岩同侵入岩相接——即矽卡岩环境。此处的接触带可赋存铁、铜，偶尔还有刚玉等宝石矿物。

寻找：石灰岩与火成岩的分界、沿线的异常矿物生长、晶形完好的晶体。

砂拉越 — 石隆门金矿区及其他

东马来西亚的地质与半岛差异很大，且有真实的黄金历史。石隆门（Bau）位于古晋附近，自 1800 年代起即有开采。矿化与侵入体侵入石灰岩和沉积岩有关——属矽卡岩型与卡林型系统，而非半岛那种花岗岩控制的石英脉。

砂拉越位于婆罗洲形成过程中较年轻、构造更复杂的地体上，因此矿床类型不同。

寻找：侵入体–石灰岩接触带、蚀变褪色岩石、铁帽、硫化物、靠近侵入体的石英脉。

沙巴 — 马穆特与斑岩区

京那巴鲁山附近的 马穆特（Mamut） 曾是大型斑岩铜金矿床，开采数十年直至 1990 年代。斑岩系统又是另一类型：以侵入体为中心的大规模、较低品位矿体，周围环绕大面积蚀变带。

寻找：大范围蚀变带（常可在卫星黏土／蚀变图层上看到）、铜染（绿／蓝）、浸染状硫化物而非窄脉，以及网脉——密集交错的细石英脉网。

⚠ 这决定了你该勘察什么

在彭亨与在砂拉越勘探，要找的东西并不相同。

• 半岛——锡–花岗岩省。靠近花岗岩的石英脉、冲积锡、红土。目标是窄而品位较高的脉体。
• 东马来西亚——斑岩与矽卡岩区。大面积蚀变带、侵入体接触带、铜金共生。目标更大但品位较低。

卫星图层在两地都有用，但解读方式不同：半岛看细窄线性构造和小片铁染，沙巴／砂拉越看宽阔的蚀变晕。

确认你伙伴的执照涵盖你所处的具体区块。矿权归州属机关，各州主管单位不同：

• 马来西亚半岛——州土地矿务局局长，技术部门为 JMG（矿物及地球科学局）。
• 砂拉越——拥有自己的矿业法规与土地行政；许可须经砂拉越州政府机关，不走联邦途径。
• 沙巴——同样通过州属机构管理自己的矿权。

在执照范围外取样可能构成违法。由于东马来西亚的规定与半岛确有不同，请在当地核实，勿凭假设——在做任何超出"看一看"的事情之前，向相关州属部门确认现行规定。

保密的同时确保安全

独自踏勘寻找目标本身没问题，但务必告诉一位信任的人你当天的路线与预计返回时间。偏远地点、河流和旧矿坑确实危险——保密绝不应意味着没有人知道你在哪里。

当你有所发现时

不要过度采集或开始任何土方作业。记录位置与照片，取一个代表性样品，送往有资质的实验室（JMG 或商业化验所）。结果为正 → 再聘请持证勘探地质师，并与你的伙伴一同遵循官方申报程序。
